Lee's Bento(李的便当)项目介绍
JiazhengHE Lv1
2021-05-26 12:12:57

注⚠️：以下内容（数据）仅基于2017年末，可能与目前有所出入，仅供参考。

1.项目介绍

1.1项目背景

目前，我国在校大学生已超3000万，高校市场容量大、分布集中且相对封闭的特点，对进入校园市场的品牌推广十分有利。但是另一个方面，相对已进入社会独立生活的消费者，大学生个体的消费能力有限、消费行为有许多不同之处。

根据《2016中国校园市场发展报告》（以下简称《报告》），2016年全国大学生月均生活费1423元，其中，高生活费地区主要集中在北上江浙等发达和东北地区。值得注意的是来自《腾讯事实说》的数据表明，仅在5年前，有接近40%的大学生月生活费不足500元，超过**75%**的大学生生活费在1000元以下。说明大学生的生活费是处于相对稳步上涨趋势的，原因除了物价水平的增高，还有其消费产品种类的改变与多样化。《2016年中国大学生消费结构》显示，三餐占比达到32%，这部分的全国基础消费达到2193亿元。

项目初期以独墅湖高教区为试点，由苏州工业园区管委会打造的独墅湖高教区，截止2017年3月，入住院校20余所，在校大学生人数已达7.6万人。

独墅湖高教区入住院校20余所，上图显示的是高教区中心商圈附近的大体量院校。

校园商圈：文星、文荟及文萃广场，翰林邻里中心。

文星广场是其中发展最好的，文萃虽然靠近2号线地铁口，硬件设施最好，但文萃宿舍区学生入住率较低，商业广场体量明显大于消费群体，故生意较差。

苏大采取的是全封闭式管理，宿舍区设置在校内，多个食堂口碑不错。

人大中法学院在校大学生约2000人，且无校园食堂，学生三餐以附近商业广场和外卖为主。

西交利物浦大学在校大学生约1万，食堂均价每餐22元，且普遍反映口味一般，根据问卷有78%的学生每天至少会点一次外卖。

高博国际学院于百年技术学院为高职院校，共计约7000人。

以《饿了么》外卖平台数据为基础，以高教区校园中心商圈文星广场为原点，辐射5公里范围，对高教区外卖数据进行相关分析：

5公里范围内645户商家，其中人均价格20-40元之间占52%。

并对户商家进行月销量与人均消费额统计:

  • 人均消费主要区间集中在20-32元。
  • 排除新店后，在20-32元消费区间内，商家月销量平均为1527单。
  • 离群值主要为爆款减脂健康餐和连锁大品牌餐饮。

同时对周边645家商户中售卖简餐便当的324家店铺经营品类进行分类，数据如下：

简餐便当以各类特色盖浇饭为主，17.6元左右，月均销量1648单。食材以鸡肉为主，口味多为咸鲜辣香的改良版川菜。其中63%为小品牌外卖加盟店，使用中央厨房生产的成品料理包与小门店简易加工相结合的方式经营；37%为现炒盖浇饭，由于现炒会对其产量有严格限制，故其月均销量低于前者，仅为1103单。

精品便当占比仅16%，均价27.2元，月均销量1602单，讲究荤素小菜饮料的搭配。便当主题多样化，有减脂增肌、日韩料理、创意西餐等。

综上所述：

  • 用户在每餐消费15-29元之间对价格不敏感，从均价17.6的特色盖浇饭到均价27.2的精品便当销量并没有出现明显的下降趋势，客单价与销量在此区间内边际递减率较低。
  • 在精品便当类中，可以将市场再次拆分为两个价格区间：其中21-24元以减脂餐，烤鸡类“高端”盖浇饭为主，口味相对统一，消费者认知度较高，月销售稳定，加盟店模式居多。25-29元区间主要以和风为主，食材摆脱了鸡肉，以牛肉，鱼类为主。
  • 对于精品便当，关键在于高认知度的菜品+合理的价格区间。

1.2 用户消费行为分析

个性、新颖：大学生群体消费“前卫”，对新兴产品的接受度远超一般消费者。

自我意识强：文娱消费潜力正在释放，整体呈现轻度娱乐型消费占据主导的倾向。娱乐型消费的特点是注重精神层面的消费升级，改善性消费。

感性：容易受身边同学影响，攀比心理导致冲动消费。特别是对情感的追求，会直接刺激物质的消费投入。

懒：便捷性和随时性，成为左右消费行为的重要因素。

纯消费者：《报告》显示，93.2%的大学生仍然需要家庭提供生活费，因此对产品和服务的优惠福利会更加关注。

1.3 项目定位

本项目旨在打造“新轻食”外卖餐配一体化，通过产品的健康及颜值在年轻群体中取得认同感，尤其是满足年轻女性群体对于生活“仪式感”的追求。不仅在产品品质上区别于一般外卖“小作坊”，在配送上也不同于传统外卖平台依靠骑手连通商家和用户的方式，而是采用订单预定，中央厨房直接完成制作，并由配送车直接配送制人流量集中区域的外卖柜，完成“最后一公里”。

2. 产品介绍

2.1 产品概要

结合以上背景及消费者行为，本项目将推出两条产品线：

注⚠️：以下图片均为产品实拍

2.1.1 日常系列

产品特点：

健康：

  • 遵循少油少盐的原则，严格控制热量，除荤菜外其他配菜均使用水煮或生食的简单烹饪方式
  • 主食中添加粗粮，均衡膳食
  • 一荤多素，每份便当添加3-5种素菜不等，种类繁多拒绝单一
  • 荤菜采用用户认可度较高的改良川菜为主，如粉蒸肉，红烧排骨，咖喱鸡等

产品定位：

中端走量款，客单价较低利润较低，其中选取一到两款用户认可度最高的产品作为爆款，配合常年折扣做到薄利多销。

去精细化：餐品对厨艺要求极低，后期全部可由中央厨房完成。

2.1.2 品质系列

产品特点：

颜值：

  • 产品呈现形式创新，通过添加可爱饭团使产品自带话题属性。
  • 产品内容创新，将中式简餐和日式饭团相结合，配合蔬菜沙拉，以“颜值”为基础，兼顾健康。

产品定位：

高端推广款：由于卡通饭团的标准化难度较高，采取每日限量供应，以减少成本；且产品自带流量，用户自发传播意愿较强。

2.2 产品用户定位

主要用户群体为较为关注食品安全和健康，且对新鲜事物有较高好奇心的在校大学生，在职白领等。前期试错阶段以大学生群体为主，后期主要用户群体以白领群体为主。用户主要消费场景集中在人群较为集中的宿舍，图书馆，写字楼，公寓等。

3 产品运营

本产品运营主要为以下两个部分。

3.1 试错阶段（已完成）

用户平台：微店：用户下单平台。 微信个人号：活动宣传平台。

主要用户群体：西交利物浦大学学生

推广方式：前期通过朋友圈，校内论坛，校园微信群等方式进行第一波活动预热，活动内容为“仅需1元吃午饭”，通过添加微信并转发朋友圈即可参与活动。外卖采用预定制出售，且60%产品为品质系列，40%为日常系列。通过产品“颜值”取得用户好感，同时由于产品自带的话题属性使得用户有自发性推广的意愿（发朋友圈炫耀）。

一期阶段成就：项目试错累计持续50天，个人微信号累计添加用户4200余人，用户转化率36%，用户总复购率61%，7日内复购率24%。

一期调研结果：本期样本容量共计504人

本期调研主要表明以下问题：

  1. 由于本阶段生产能力有限，经常出现供不应求，很多用户因为没有办法“抢”到外卖而放弃下单。
  2. 菜品质量仍需改进，但好吃和健康是完全相悖的两个概念，因此还需要探索用户的平衡点已提升用户体验。
  3. 受访者有67.2%反应对社区手帐等ip文创很感兴趣，且用户主要以女性群体为主，因此在后期可以着重建设社区版块。

3.2 孵化阶段 （部分实现）

用户平台：前期与4.1相同，后期以微信支付宝小程序（公众号）为主。

主要用户群体：独墅湖高教区内大学生

其中

公众号二次开发，主要实现以下功能：

  • 订餐：客户在线下单，选择配送要素；收藏，产品降价后自动通知；产品及折扣页面设置分享按钮，一键朋友圈和微博转发。
  • 用户管理：会员个人消费数据、等级福利等展示。
  • 社区：用户可发布手帐等ip文创作品，同时具备收藏，点赞，评论等功能；公众号每周精选高赞藏作品作为推文集合发布。
  • 用户消费数据统计功能：建立客户群组来衡量指标：访问间隔，点击内容等。

销售方式改变：从传统外卖的随点随吃到预定制转变，将客户分流。

核心客户：对核心客户采用周订餐方式（客户需在服务号及小程序下单），即一次下单预定一周工作日午饭，外卖采用高低搭配，根据价格可选择每周1至2顿品质系列，其他为日常系列。用户仅需要标注忌口即可，免去选择困难的烦恼。

新客户与非核心客户：走传统外卖平台，品质系列限量供应，日常系列每周选取一两款打折。

配送方式的改变：从点对点到点对面。

对与核心客户工作日午餐，采用集中配送至人流密集区域恒温外卖柜的方式。

对于非核心客户依旧采用平台自有外卖骑手进行配送。

两期累计项目收益：总计收益GMV10W，累计净利润收益3W，单日产品售罄清仓时间记录最短达5min，投放后得到一致好评，用户转化率达20.5%，七日内复购率达18.3%，总复购率达53%。

4.写在最后

关于为什么没有做下去，其实还是很现实的原因，没钱。关于外卖柜的想法来源于小区里的社区生鲜品牌“食行生鲜”，冷链柜在17年可以说已经烂大街了，打开阿里巴巴搜恒温柜98%都是冷链柜，好不容易找到一家常熟的工厂可以做热链恒温柜，去参观了一趟谈了下价格还是”打扰了“，毕竟我们3个合伙人在学校旁边租了间100平米的小厂房再买了一堆食品加工设备也就花了不到4万，而当时一台96格的柜子加上后期服务就差不多10万，现实始终是难以跨越的一道鸿沟，况且即使买了柜子，按照二期规划开始的利润，回本也遥遥无期，就像陷入了无底洞，想挣钱就要扩大规模，想扩大规模就要投入，不然怎么说万物起源是“鸡生蛋还是蛋生鸡”问题呢。我始终相信，好的生意都是长期投资的，给自己的“小目标”也很简单，收益率能打败通货膨胀就行了。时至2021，看见学校和宿舍楼下屹立的美团外卖柜，想想自己也有过相同的idea，也在内心为自己小小的自豪了一把，当然我也很清楚，一个idea到落地实施，这中间需要无数的试错，论证，以及良好的后期运营，这中间的环节缺一不可，即使我走对了一步，接下来还有无数个坑等着我。当然这仅仅算是一次勇敢的尝试，其实作为一个大学生，我从来没有把盈利当作第一要义，而是能把想法在自己的能力范围内实现，勇敢的走出去这一步，让自己多感受，多学习，并将其作为一种伴随一生的经历至少是值得的。
