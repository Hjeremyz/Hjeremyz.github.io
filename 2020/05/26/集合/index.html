集合

简介：集合是用来储存多个元素的容器

集合和数组的区别：

元素类型：

  • 集合：引用类型（储存基本类型是自动装箱）
  • 数组：基本类型，引用类型

元素个数：

  • 集合：不固定，可任意扩容
  • 数组：固定，不能改变容量

集合的好处：不受容器大小限制可以随时添加删除元素，提供了大量操作元素的方法（判断，获取等）

java的集合体系：

  • 单列集合：Collection: 1. List ->ArrayList 2. Set ->HashSet
  • 双列集合：Map->HashMap

List集合特点

可重复，有序（存取顺序相同）

使用步骤：

  1. 创建集合对象
  2. 创建元素对象
  3. 将元素对象添加到集合对象
  4. 遍历集合对象

增强for循环：简化数组和集合遍历

  • for（数据类型 变量名： 数组或集合对象）{

迭代器

为什么需要迭代器？

对过程的重复称为迭代。

迭代器是遍历Collection集合的通用方式，可以在对集合遍历的同时进行添加，删除等操作。

迭代器的常用方法

  • next():返回迭代的下一个元素对象
  • hasNext():如果仍有元素迭代则返回true

迭代器使用步骤：

  1. 根据集合对象获取其对象的迭代器对象
  2. 判断迭代器中是否有元素
  3. 如果有就获取元素

案例演示：使用迭代器遍历list集合

注意：普通迭代器在遍历的同时不能添加或删除元素，否则会报错：并发修改异常，如需添加，请用ListIterator（列表迭代器）.

泛型

概念

即泛指任意类型，又叫参数话类型(ParameterizedType),对集体类型的使用起到辅助作用，类似于方法的参数。

集合类泛型的解释

表示改类集合中存放指定类型的元素

泛型的好处

类型安全； 避免了类型转换

Collection工具

针对集合操作的工具箱

成员方法：

  • sort(list):根据元素的自然顺序，将指定列表按生序排列
  • max(Collection):返回集合的最大元素
  • reverse(List):反转list集合元素
  • shuffle(List):使用默认的随机元随机置换指定的列表（洗牌功能）

Set集合的特点和应用

特点：不可重复，无序

应用：Set set = new HashSet<>();

Example:向set集合中添加3个元素，并遍历打印

Map集合的特点和应用

特点

双列集合，元素由key值对（Enter）构成

key–value

特点：无序、以键值对的形式添加元素，键不能重复，值可以重复

应用：

Map<T1,T2> map = new HashMap<>();

Example:向Map中添加三个元素，并遍历打印

分析：一

  • 向Map中添加元素的方法：put()
  • 遍历所有集合的方式： 1.获取所有的key：keySet() 2.遍历KeySet,通过key获取value:get() 3. 遍历方法：iterator()

成员方法：

  • v put(K key, V value):添加元素，元素第一次添加，返回null，重复添加，会用新值覆盖旧值，返回旧值。
  • get()
