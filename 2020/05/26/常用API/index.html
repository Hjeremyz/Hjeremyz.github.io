常用API

java中组建的层次结构：Module->Package->Class

Object类

属于java.base->java.lang->Object

所有类都是objetc类的对象，所有类都直接或间接继承object类。

构造方法

  • Object()

    构造一个对象。所有的子类对象初始化都会优先调用这个方法

成员方法

  • Int hashCode()

​ 返回对象的哈希码值，该方法通过对象的地址值进行计算，不同对象的返回值一般不同。

  • Class<?> getClass()

​ 返回调用方法对象的运行时类对象（调用者的字节码文件对象。一个类对应的字节码文件只有一个。

  • String toString()

​ 返回该对象的字符串表示，默认打印地址值。地址值的组成格式：全类名+@+哈希码无符号十六进制

  • Boolean equals()

​ 返回其他狗个地域性是否与此对象相等。默认情况比较两个对象的引用，建议重写。

注意：java.lang包下的类不用导包直接使用。

Scanner类

属于java.base->java.util->Scanner->Scanner(InputStream)

String类

属于java.base->java.lang->String

简介：字符串，每一个字符串对象都是一个常量。

构造方法：

  • String(byte[]):构造一个string对象，将指定字节数组中的数据转化成字符串。
  • String(char[]):构造一个string对象，将指定字符数组中的数据转化成字符串。

成员方法

判断方法

  • boolean equals(String)
  • boolean eaualsIgnoreCase(String)
  • boolean startsWith(String):判断是否以给定字符串开头

获取方法

  • Int length():获取当前字符串长度
  • Char charAt(int index):获取指定索引位置的字符
  • int IndexOf(String):获取指定字符串第一次出现的索引
  • int lastIndexOf(String):获取指定字符串最后一次出现的索引
  • String substring(int):截取(获取)指定索引位置(含之后)的字符串
  • String substring(int, int):获取从索引start位置到end位置的字符串(左闭右开区间)。

转换方法

  • Byte[] getBytes():将字符串转换成字节数组
  • char[] toCharArray():将字符串转换成字符数组
  • Static String valueOf(..):将指定类型数据转换成字符串
  • String replace(old, new):将指定字符串替换成新字符串
  • String[] split(String):切割字符串，返回切割后的字符串数据，原字符串不变
  • String trim():去除字符串两端的空白字符

StringBuilder和StringBuffer

简介：可变字符序列，用于构造字符对象。内部使用自动扩容的数组操作字符串数据。StringBuilder和StringBuffer使用相同的api。

属于java.base->java.lang->StringBuilder&StringBuffer

Date类和Calendar类

属于java.base->java.util->Date/Calendar

简介：日期和日历类，用于操作日期相关信息。

构造方法：

  • Date():构造日期对象，当前系统时间，精确到毫秒
  • Date(long):构造日期对象，时间自“1970年1月1日 00:00:00 GMT”起，至指定参数的毫秒数

成员方法：

  • Long getTime():将日期对象转换成对应的时间毫秒
  • static Calendar getInstance():根据当前系统时区和语言环境获取日历对象
  • Int get(int field):分会给定日历字段的值
  • void set(int field, int value):将给定的日历字段设置为指定的值

基本类型的包装类

属于java.base->java.lang->包装类

简介：基本数据类型不是对象，所以Java针对基本类型提供了对应的包装类，以对象形式来使用。

使用步骤：

  • 装箱：基本类型转成包装类型。
  • 拆箱；包装（对象）类型转成基本类型

成员方法：将字符串类型的数据转换成对应的基本类型

  • static 基本类型 paresXXX(String):将字符串类型转换成基本数据类型
